# Tovade's api

## How to use

1. `git clone https://github.com/tovade-dev/api`
2. Rename config.example.js to config.js
3. Fill in config.js
4. npm start
   And your version of the api should be online!
   Not working? Join my [discord](https://discord.gg/u8NgtppkRp)!
